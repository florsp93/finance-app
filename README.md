Finance App es una web app que desarrolle con React y Bootstrap.
Al ser una progressive web app, se comporta en algunos aspectos como una app nativa, es posible agregarla a la pantalla de inicio en el celular, utilizarla sin conexión a internet, y también nos permite cerrarla y no perder los datos que fuimos ingresando.  
Para acceder → http://finance-app.wappstudio.com/
